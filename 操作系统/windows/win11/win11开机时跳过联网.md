# win11 装机时跳过联网

**操作方法 **

　　首先，进入系统安装选择界面。

![win1101.png](/image/win/win1101.png)

进入后，键盘按住`shift+F10`，弹出`cmd`命令框，然后输入**`OOBE\BYPASSNRO`**回车。
```bash
OOBE\BYPASSNRO
```

![win1102.png](/image/win/win1102.png)

　　等待系统重启，在提示联网界面选择【我没有Internet连接】.
