### 搭建域控服务器

#### 前置准备

1、 开启win server 远程访问

此电脑  =>   右键 => 属性 => 远程设置 

![image-20240703103135192](01_域控服务器选型搭建.assets/image-20240703103135192.png)

2、固定IP 地址和dns 地址和时间服务器

![image-20240703103343718](01_域控服务器选型搭建.assets/image-20240703103343718.png)

3、修改计算机名字

此电脑  =>   右键 => 属性
